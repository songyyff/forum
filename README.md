## 众言论坛

众言论坛是基于Apache服务器，PHP和javascript脚本语言并以MySQL为后台数据库服务的网络论坛平台。它可以架设在Linux, Unix, Windows任何一种系统上。论坛运行还需要Apache,Php和MySQL三个基础服务相配合。

# 合适的使用范围
论坛采用积木式设计结构，具有不同功能的组合模块，使用范围广泛，可方便的作为以下目的使用：

1.建设网络论坛
  可以很好的支持各种类型的网络论坛。这类论坛的网络流量和服务器性能要求比较高，论坛架设可以使用租用空间，但最好使用专用服务器。

注：对租用空间的论坛安装有些不同，您可以在论坛安装部分查看安装方法。

2.公司内部论坛
  可方便的把本论坛架设在安装有Apache，PHP和MySql的Windows或Linux的平台，建立公司内部开放的知识和信息交流平台是一个很好的选择。

3.社区论坛
  社区论坛的建设与小型网络论坛非常相似，性能需求接近。

论坛可以放置在租用空间中，也可以放在社区自己的服务器内。

# 论坛特点
在系统设计上，论坛有如下特点：

1.便捷的安装
  论坛的架设不是只有论坛程序本身，它需要几个程序的相互配合和配置，并且要求安装人员具有一定的网络知识；为了降低论坛安装的难度，我们特别编写了简便的安装程序和安装指南；因此，您只需要轻松的按照安装指南即可快速的架设好论坛。

2.友好的界面
  直观而友好的界面，清楚的把相同功能组合在一起，使用户快捷的达成目的；并设有不同的主题可供您选择。

3.分级的权限结构
  论坛的的权限有三类，用户权限，论坛权限，管理员权限；他们相互限制着您在论坛的行为，如果您想确切了解这部分的内容，可以查阅论坛的管理员帮助。

4.清晰的功能分类
  论坛按功能的区别，把短消息，用户资料，通用模块，查询系统，帮助系统，管理员操作分别聚合在一起，以方便操作。

5.完善的帮助系统
  可根据您的论坛身份，是游客，普通用户，还是论坛管理员，显示出适合您的帮助内容。

6.全面的管理系统
  管理员可以对用户，具有管理权的论坛，论坛里的帖子，回复执行修改内容和权限，失有效等管理操作；管理员甚至可以定制自己的管理信息。
